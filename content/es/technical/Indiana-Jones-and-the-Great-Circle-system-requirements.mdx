<Section icon={Shield} title="Requisitos mínimos: 1080p a 60 FPS">

Un aspecto refrescante de los requisitos del sistema de Indiana Jones and the Great Circle es la tasa de fotogramas objetivo. La mayoría de los desarrolladores listan especificaciones mínimas para 30 FPS, pero aquí, la base es unos sólidos 60 FPS a 1080p usando el ajuste Bajo. Curiosamente, este objetivo se logra a **Resolución Nativa**, lo que significa que no necesitas depender del escalado (DLSS/FSR) para alcanzar velocidades jugables en hardware moderno de nivel de entrada.

El suelo de la GPU se establece en la RTX 2060 Super o la RX 6600. Es vital notar la designación "Super" para Nvidia; la RTX 2060 estándar con 6 GB de VRAM puede tener dificultades debido a la alta carga de memoria de los activos con trazado de rayos. El juego requiere 8 GB de VRAM como base estricta para la estabilidad.

| Componente | Especificación mínima (1080p/60) |
| :--- | :--- |
| **SO** | Windows 10/11 (64 bits) |
| **CPU** | Ryzen 5 3600 / Intel Core i7-10700K |
| **RAM** | 16 GB |
| **GPU (Nvidia)** | RTX 2060 Super (8GB) |
| **GPU (AMD)** | Radeon RX 6600 (8GB) |
| **Almacenamiento** | 120 GB SSD |

<Callout type="tip" title="Gestión de VRAM">
Si tienes una tarjeta de 8 GB, evita ejecutar aplicaciones en segundo plano como Chrome o Discord con la aceleración de hardware activada, ya que esto puede consumir la VRAM necesaria para los cálculos de trazado de rayos del juego.
</Callout>

El requisito de CPU aumenta significativamente en el nivel Recomendado, pasando de un Ryzen 3600 a un Ryzen 7700 o i7-12700K. Esto sugiere que los ajustes gráficos más altos o los entornos más densos ponen una carga mucho más pesada en la capacidad del procesador para manejar las llamadas de dibujo y las estructuras RT BVH.

<Section icon={Target} title="La experiencia completa de trazado de rayos (Path Tracing)">

Para el entusiasta que quiere que cada rebote de luz se calcule en tiempo real, está disponible el modo de "Trazado de rayos completo", a menudo denominado Path Tracing. Este modo es increíblemente exigente y está diseñado teniendo en cuenta las características de la serie 40 de Nvidia, específicamente Frame Generation y DLSS 3.5 Ray Reconstruction.

Para alcanzar los 60 FPS en este modo, los desarrolladores asumen el uso de Super Resolution y Frame Generation. Sin estas tecnologías, incluso una RTX 4090 puede tener dificultades para mantener una tasa de fotogramas constante a 4K.

<Steps>
  <Step num="1" title="Verificar hardware">
    Asegúrate de tener una GPU de la serie RTX 40 para utilizar Frame Generation, que es esencial para los objetivos de Full RT.
  </Step>
  <Step num="2" title="Activar DLSS/FSR">
    Establece la Super Resolution en 'Calidad' para 1080p o 'Rendimiento' para 4K para equilibrar la carga del path tracing.
  </Step>
  <Step num="3" title="Activar Frame Gen">
    Activa Frame Generation en el menú de ajustes para cerrar la brecha hacia una salida visual de 60 FPS.
  </Step>
  <Step num="4" title="Reconstrucción de rayos">
    Activa DLSS 3.5 Ray Reconstruction para mejorar la claridad de los reflejos y la iluminación global.
  </Step>
</Steps>

<Callout type="info" title="Requisitos de Full RT">
El modo 4K Full RT apunta a 60 FPS usando el **Modo de rendimiento DLSS** y **Frame Generation** en una RTX 4090. Este es el ajuste más exigente disponible actualmente en los juegos modernos.
</Callout>

</Section>
