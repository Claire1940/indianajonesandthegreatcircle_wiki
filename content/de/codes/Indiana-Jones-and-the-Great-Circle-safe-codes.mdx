Der erste Safe, dem Sie begegnen werden, befindet sich im **Belvedere-Hof**. Wenn Sie den offenen Bereich erreichen, suchen Sie direkt vor sich das erste Militärzelt. Auf dem Schreibtisch darin finden Sie einen Ticketabschnitt. Indem Sie die Vorder- und Rückseite des Abschnitts inspizieren, können Sie den vierstelligen Code zusammensetzen.

| Ort | Hinweisobjekt | Safe Code | Belohnung |
| :--- | :--- | :--- | :--- |
| Belvedere Hof Zelt | Ticketabschnitt | **5238** | Kampffähigkeitsbuch, Geld |
| Postkeller | Notiz des Wächters | **1136** | Questfortschritt, Beute |

<Callout type="warning" title="Feindliche Präsenz">
Der Belvedere-Hof wird stark patrouilliert. Stellen Sie sicher, dass Sie die unmittelbaren Wachen beseitigen oder Tarnung verwenden, bevor Sie versuchen, den Ticketabschnitt auf dem Schreibtisch zu inspizieren, da das Entdecktwerden die Inspektionsanimation unterbricht.
</Callout>

Ein weiterer wichtiger Ort ist der Keller unter dem Postamt, in der Nähe des **Nikolausturms**. Dies ist Teil der Feldforschung **Wilde Entdeckung**. Sie müssen zuerst den Kellerschlüssel von einem nahegelegenen Feind erhalten. Sobald Sie drinnen sind, schalten Sie den Wächter aus und lesen Sie die Notiz, die er bei sich trägt. Die Notiz erwähnt "November letzten Jahres". Da das Spiel im Jahr 1937 spielt, bezieht sich "letztes Jahr" auf 1936, und November ist der 11. Monat, was Ihnen den Code **1136** gibt.

<Section icon={Target} title="Museums- und Gartenrätsel">

Die Feldforschung **Eine Nonne in Not** führt Sie zu einigen der lohnendsten Safes im Spiel. Diese erfordern eine Kombination aus Kalenderlesen und Umweltinteraktion.

  <Card title="Museums-Garten-Safe">

  </Card>
  <Card title="Museumskeller-Safe">
    Wird während der Museumsinfiltration gefunden. Dieser Safe ist einzigartig, da der Code durch Beleuchtung sichtbar versteckt ist.

Um den Gartensafe zu öffnen, müssen Sie den "Geheimen Lieferbrief" finden, der besagt, dass ein Paket am 15. Oktober angekommen ist. Wenn Sie den In-Game-Kalender überprüfen, sehen Sie, dass der 15. Oktober auf einen **Freitag** fällt. Ein Abgleich von "Freitag" mit den Würfelsymbolen auf dem Kalender ergibt den Code **6380**. Im Inneren finden Sie ein Buch, das alle Medizinstandorte im Vatikan detailliert beschreibt.

<Steps>
  <Step num="1" title="Den Keller-Safe finden">
    Betreten Sie den Museumskeller während oder nach der Quest "Eine Nonne in Not".
  </Step>
  <Step num="2" title="Mit dem Schreibtisch interagieren">
    Finden Sie den Schreibtisch mit der Lampe neben dem Safe.
  </Step>
  <Step num="3" title="Das Licht ausschalten">
    Interagieren Sie mit der Lampe, um sie auszuschalten. Die Schatten/das Leuchten enthüllen den Code **7171**, der in die Schreibtischoberfläche eingraviert ist.
  </Step>
</Steps>

<Callout type="tip" title="Schnellreise-Tipp">
Auch nach Abschluss der Feldforschung können Sie schnell zum Gartenstandort zurückreisen, um auf den Museumskeller zuzugreifen, falls Sie den Safe bei Ihrem ersten Durchlauf verpasst haben.
</Callout>

<Section icon={Sparkles} title="Sixtinische Kapelle und Mysterien-Safes">

Die Sixtinische Kapelle beherbergt zwei verschiedene Safes, die beide mit "Mysterien" verbunden sind, die einige der besten narrativen Belohnungen und Fähigkeitsverbesserungen im Spiel bieten.

Das Mysterium **Geheimnis der Geheimnisse** wird im Hinterzimmer der Sixtinischen Kapelle ausgelöst. Sie finden eine Notiz, die Sie anweist, die Statuen von **Petrus (mit dem Schlüssel)** und **Paulus (mit dem Schwert)** zu inspizieren. Das Umdrehen dieser Statuen enthüllt Planetensymbole. Sie müssen dann die Planetenkarte im Raum finden und die Planeten nach ihrer Entfernung von der Sonne ordnen.

| Planetensymbol | Reihenfolge von der Sonne | Resultierende Ziffer |
| :--- | :--- | :--- |
| Symbol Statue 1 | 4. Position | **4** |
| Symbol Statue 2 | 4. Position | **4** |
| Symbol Statue 3 | 7. Position | **7** |
| Symbol Statue 4 | 1. Position | **1** |

Der endgültige Code für dieses komplexe Rätsel ist **4471**. Dies schaltet ein Mysterienbuch und ein zusätzliches Fertigkeitsbuch frei. Später, nach Abschluss der Nonnen-Questreihe, können Sie für **Ginas Anfrage** (Anzeichen von Ärger) zurückkehren. Dieser Safe befindet sich in einem einzelnen Gebäude an der Ausgrabungsstätte. Der Code, **1891**, ist deutlich am unteren Rand von Ginas Notiz geschrieben.

Das Abschließen dieser Safe-bezogenen Mysterien trägt erheblich zu Ihrem Gesamt-Abschlussgrad für den Vatikan-Bezirk bei.
</Callout>

</Section>

<Section icon={Swords} title="Die Vater-und-Sohn-Chiffre">

Der vielleicht komplizierteste Safe ist mit dem Mysterium **Vater und Sohn** verbunden. Dieser Safe befindet sich in der Nähe des **Borgia-Turms** in einem Nebenraum, der Julianas Schlüssel zum Betreten erfordert. Im Inneren finden Sie eine Katze, einen Hund und ein kryptisches Gedicht.

Um dies zu lösen, müssen Sie das Gedicht plündern und auf die eingekreisten Buchstaben achten: **I, R, I, A, T, A, S, O**. Sie müssen diese Buchstaben dann mit einem im Raum gefundenen Quadratdiagramm vergleichen. Jeder Buchstabe oder jede Buchstabenkombination entspricht einer bestimmten Zahl im Raster.

| Buchstabengruppe | Gitterkoordinate | Resultierende Ziffer |
| :--- | :--- | :--- |
| I / R | Reihe 5 | **5** |
| I / A | Reihe 8 | **8** |
| T / A | Reihe 7 | **7** |
| S / O | Reihe 3 | **3** |

Der Reihenfolge der eingekreisten Buchstaben des Gedichts folgend führt zum Code **5873**. Das Entsperren dieses Safes schließt das Vater-und-Sohn-Mysterium ab und gibt einen erheblichen Schub für Ihre Abenteuerpunkte.

<Callout type="success" title="Mysterium gelöst">
Sobald Sie **5873** eingeben, öffnet sich der Safe und belohnt Sie mit einzigartigen Lore-Gegenständen und beendet eines der schwierigsten Rätsel im Vatikan.
</Callout>

</Section>
