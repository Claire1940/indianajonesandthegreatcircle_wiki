  </Step>
  <Step num="2" title="Ersten Drachen drehen">
    Drehen Sie die Klaue, um einen Hebel freizulegen. Benutzen Sie diesen Hebel, um die erste Drachenstatue so auszurichten, dass sie der anderen Drachenstatue auf der gegenüberliegenden Seite des Brunnens zugewandt ist.
  </Step>
  <Step num="3" title="Zweiten Drachen erreichen">

  <Step num="4" title="Zweiten Drachen ausrichten">

Dieser Abschnitt ist eine klassische "Objekt bewegen"-Herausforderung, die Scheiben und eine Bootsfigur beinhaltet. <Callout type="success" title="Rätselübersicht">
Das Ziel ist es, eine Bootsfigur von der linken Seite zur rechten Seite zu bewegen, indem zwei Scheiben (eine kleine und eine große) in der Mitte so ausgerichtet werden, dass ein durchgehender Weg entsteht.
</Callout>

Nach Abschluss der Taufszene drehen Sie die Struktur erneut mit dem Hebel, um die andere Seite freizulegen, die das Rätsel des Bogens von Basch beherbergt. Dieser Teil zeigt ein Kreuz und mehrere Scheiben. Die Kernlösung besteht darin, eine **Bootsfigur** von der linken Seite des Rätselbereichs zur rechten Seite zu führen. Dies wird erreicht, indem die beiden Scheiben – eine kleine und eine große – in der Mitte so ausgerichtet werden, dass ein durchgehender Weg entsteht.

Dann verwenden Sie einen Ring in der oberen rechten Ecke (wahrscheinlich mit Ihrer Peitsche), um das Boot in den mittleren Abschnitt zu bringen. Sobald sich das Boot in der Mitte befindet, können Sie den Weg auf der rechten Seite freimachen, damit das Boot sein endgültiges Ziel erreichen kann.
