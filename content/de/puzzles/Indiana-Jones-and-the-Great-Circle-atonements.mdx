<Steps>
  <Step num="1" title="Erste Aufklärung">
    Sobald Sie die Kontrolle übernommen haben, machen Sie sofort ein Foto vom **Ausgrabungsbüro**, in dem Gina festgehalten wird. Dies ist eine Abenteuer-Notiz. Rutschen Sie von Ihrem Aussichtspunkt nach unten.
  </Step>
  <Step num="2" title="Wählen Sie Ihren Einstiegspunkt">
    Sie haben zwei Hauptwege, um das Lager zu betreten:
    - **Rohrtunnel:** Rechts vom versiegelten Tor, hinter überdachten Lastwagen, finden Sie eine Luke, die zu einem Rohrtunnel unter dem Hundezelt führt. Hier finden Sie eine **Mersu-Rezept** Abenteuer-Notiz. Dieser Weg führt durch eine Straße mit vielen patrouillierenden Wachen.
    - **Gerüstgrube (Empfohlen):** Links vom Haupteingang befindet sich eine große Grube mit wackeligem Gerüst. Springen Sie schnell darüber, da die Plattformen bröseln. Auf der anderen Seite finden Sie ein kleines Gebäude mit einer **Ausgrabungslagerkarte** und einer Pistole. Dieser Weg bietet bessere Tarnmöglichkeiten.
  </Step>
  <Step num="3" title="Navigieren Sie das Lager heimlich">
    Nachdem Sie über die Gerüstgrube eingetreten sind, gehen Sie nach Westen. Sie werden wahrscheinlich auf eine Wache stoßen, die um Hilfe funkt; schalten Sie ihn leise aus. Überprüfen Sie seinen Schreibtisch auf eine **Zensierte Notiz** Abenteuer-Notiz. Gehen Sie an bröckelnden Mauern vorbei zu einem Barackenzelt. Darin können Sie eine verschlossene Waffenkammer für Waffen aufbrechen. Draußen können zwei dienstfreie Wachen bei den Duschen heimlich neutralisiert werden. Überprüfen Sie die Bänke auf **Horrorgeschichten #4**.
  </Step>
  <Step num="4" title="Ginas Standort erreichen">
    Nördlich der Duschen befindet sich ein Essbereich mit einer weiteren Wache. Nachdem Sie sich um ihn gekümmert haben, nähern Sie sich der Südseite des Ausgrabungsbüros. Schalten Sie eine patrouillierende Wache aus und schauen Sie durch die Fenster. Sie können Gina vom ganz linken Fenster aus rufen, aber Sie brauchen einen anderen Weg hinein. Der Raum vor Ginas Zelle hat eine einzelne Wache, einen Offizier und eine weitere Wache an der Eingangstür. Verwenden Sie Ablenkungen oder Kampf, um den Bereich zu räumen.
  </Step>
  <Step num="5" title="Gina befreien">
    Sobald der Offizier ausgeschaltet ist, fotografieren Sie das **Zikkurat-Modell** (Abenteuer-Notiz) und finden Sie den **Lagerraumschlüssel** im zweiten Raum. Öffnen Sie Ginas Tür und befreien Sie sie. Sie wird darauf bestehen, das Abenteuer mit Ihnen fortzusetzen.
  </Step>

<Section icon={Shield} title="Aufstieg zum Zikkurat: Rätsel und Fortschritt">
Mit Gina an Ihrer Seite ist Ihr nächstes Ziel, das Zikkurat zu erreichen und seine Geheimnisse zu lüften. Dies beinhaltet das Navigieren durch weitere Nazi-Patrouillen und das Lösen von Umwelträtseln.

### Weg zum Zikkurat
Nach der Flucht aus dem Büro werden Sie das Zikkurat in der Ferne sehen, bewacht von Nazis.
- Machen Sie ein Foto vom **Zikkurat von Ur** vom vorderen Bereich aus, indem Sie sich entlang zweier Zelte im Osten schleichen.
- Halten Sie sich ganz links, um die meisten Wachen zu umgehen. Warten Sie, bis ein gepanzerter Nazi eine Brücke überquert, und bewegen Sie sich dann hinter gepanzerten Fahrzeugen, um zwei abgewandte Wachen auszuschalten.
- Klettern Sie eine zerfallene Treppe hinauf und springen Sie auf ein Gerüst. Helfen Sie Gina hoch, damit sie einen Kran bewegen kann, dem Sie folgen können.
- Oben angekommen, fotografieren Sie die **Nazi-Grabungsstätte** (Abenteuer-Notiz).
- Zwei Nazis werden einen Aufzug verlassen; Sie können ihnen ausweichen oder sie vom Sims stoßen.
- Bevor Sie hinuntergehen, finden Sie **Ausgrabungspläne** (Abenteuer-Notiz) auf einem Tisch neben einer einzelnen Wache an der Ecke der Grabungsstätte.

### Zikkurat-Innenraum
Der Innenraum des Zikkurats bietet mehrere mechanische Rätsel.

| Aufgabe | Beschreibung | Schlüsselobjekt/Aktion |
| :--- | :---------- | :-------------- |
| **Erster Aufzug** | Fahren Sie mit Gina mit dem Aufzug nach unten, aber er hält auf einer hohen Plattform. | Finden Sie das **Ninive-Dokument** (Abenteuer-Notiz) zwischen den Aufzügen. |
| **Zweiten Aufzug mit Strom versorgen** | Der zweite Aufzug ist stromlos. | Schieben Sie sich am Aufzug vorbei, überqueren Sie eine umgestürzte Leiter, seilen Sie sich zu einer tieferen Plattform ab. |
| **Eine Brücke bauen** | Erreichen Sie eine Sackgasse auf tieferen Plattformen. | Benutzen Sie die Peitsche an einer großen Holzwand, um eine Brücke zu bauen. |
| **Strom aktivieren** | Klettern Sie auf der anderen Seite eine Leiter hinauf. | Finden Sie einen Verteilerkasten neben einem Generator, um den Aufzug mit Strom zu versorgen. |
| **Hindernis beseitigen** | Der Aufzug bleibt hängen, bevor er den Boden erreicht. | Verlassen Sie den Aufzug mit der Peitsche, schwingen Sie zur anderen Seite, lassen Sie sich unter den Aufzug fallen. |
| **Endgültiger Abstieg** | Reißen Sie mit Ihrer Peitsche Holzbalken heraus, die den Abstieg des Aufzugs blockieren. | Gina schickt den Aufzug nach unten; klettern Sie hinein, um das Erdgeschoss zu erreichen. |

Unter dem Zikkurat entdecken Sie ein monumentales Geheimnis, das zum letzten Showdown mit Emmerich Voss führt.

### Der Geheime Tresor
Rechts vom Ausgang des Zikkurats führt ein großer Tunnel zum **Geheimen Tresor**.
- Fotografieren Sie den **Geheimen Tresor** (Abenteuer-Notiz).
- Dieses Rätsel erfordert **alle 50 Antiken Relikte**. Obwohl Sie zu diesem Zeitpunkt wahrscheinlich 49 haben werden, wird das letzte nach dem Spiel gesammelt.
- Sie können das Rätsel mit 49 Relikten versuchen, aber es wird empfohlen, das Spiel zuerst abzuschließen, um potenzielle Fehler zu vermeiden und sicherzustellen, dass Sie die Errungenschaft „Schatten aus der Zeit“ erhalten.

| Türbereich | Benötigte Relikte |
| :----------- | :-------------: |
| **Erste Tür** | 11 |
| **Zweite Tür** | 12 |
| **Dritte Tür** | 13 |
| **Vierte Tür** | 14 |

<Callout type="tip">
Um das Rätsel der Antiken Relikte zu lösen, bildet das Drehen eines Relikts nach links eine Box, während das Drehen nach rechts Löcher in alle vier Richtungen füllt, bis es auf eine Lücke trifft. Dies erfordert eine sorgfältige Ausrichtung.
</Callout>

### Zum Zelt und Noahs Arche gelangen
Nachdem das Geheimnis des Zikkurats gelüftet wurde, plant Indy, zu einem Zelt am Rande des Grabungsgeländes zu gehen.
- Fotografieren Sie die riesige **Noahs Arche** (Abenteuer-Notiz).
- Die Hauptstraße zum Zelt ist stark bewacht.
- Schleichen Sie an den Wachen vorbei, indem Sie den schmalen Pfad nach rechts benutzen, vorbei an einer sitzenden Wache, die Sie ausschalten können.
- Benutzen Sie Kisten und Felsen als Deckung, dann benutzen Sie Ihre Peitsche, um auf einen Vorsprung zu klettern und das bewachte Tor zu umgehen.
- Schalten Sie die letzte Wache aus, die Lastwagen inspiziert, und finden Sie die **Arche-Analyse-Notiz** (Abenteuer-Notiz) auf einer zerbrochenen Kiste in der Nähe des Eingangstors.
- Bevor Sie das Zelt betreten, decken Sie sich mit Nahrung für Gesundheit und Ausdauer ein.

### Bosskampf: Emmerich Voss
Voss wird Sie überraschen, was zu einer Nahkampfsequenz führt.

<CardGrid cols={2}>
  <Card title="Voss's Angriffsstrategien">
    - **Hackerangriffe:** Kommen aus beiden Richtungen, variierende Geschwindigkeit. Mit einer Hand blocken, mit der anderen schlagen.
    - **Karate-Kid-Kick:** Kann nicht geblockt werden. Weichen Sie während seines Vorwärtssprungs zur Seite aus, um zu kontern.
    - **Greifangriff:** Hebt beide Hände, bevor er taumelt. Schnell, schwer auszuweichen, aber schlagen Sie die Angriffstasten, um ihn abzuschütteln.
    - **Drei-Treffer-Combo:** Übergänge von Hieben, endet mit einem mächtigen Angriff. Wenn Sie den ersten nicht blocken können, weichen Sie dem letzten Treffer aus.
  </Card>
  <Card title="Kampfstrategie">
    - **Keine Waffen:** Sie haben keine Waffe oder Peitsche; verlassen Sie sich auf Fäuste.
    - **Blocken & Kontern:** Warten Sie auf Hackerangriffe, blocken Sie, dann landen Sie 3 schnelle Schläge.
    - **Ausweichen & Bestrafen:** Bei Kicks weichen Sie zu seiner Seite aus und schlagen ihn, während er sich erholt.
    - **Ausdauermanagement:** Lassen Sie die Ausdauer zwischen den Austauschphasen wiederherstellen.
    - **Letzte Phase:** Sie werden mit Voss über der Arche gefesselt sein. Treten Sie ihn weg, blocken Sie Hiebe. Gina wird eingreifen. Schauen Sie nach oben und greifen Sie ihre Hand, um zu entkommen und Voss seinem Schicksal zu überlassen.
  </Card>
</CardGrid>

<Callout type="success">
Den Kampf zu gewinnen ist nicht unbedingt notwendig, da Gina Sie letztendlich retten wird. Konzentrieren Sie sich darauf, zu überleben und den kontextuellen Anweisungen zu folgen.
</Callout>

### Den Geheimen Tresor freischalten
- **50. Antikes Relikt:** Kehren Sie zu dem Zelt zurück, in dem Voss Sie gefangen genommen hat. Es ist jetzt zugänglich. Das letzte Antike Relikt befindet sich auf dem Haupttisch durch die neu geöffnete Zelttür.
