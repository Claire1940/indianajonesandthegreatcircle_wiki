Une fois à l'intérieur de la pièce cachée, vous trouverez le coffre verrouillé ainsi que deux documents cruciaux qui détiennent la clé de sa combinaison : la note « Bases des chiffres » et le porte-documents « Code des chiffres égyptiens ». </Callout>

### Processus de déchiffrement étape par étape

<Steps>
  <Step num="1" title="Localiser les documents">
    À l'intérieur de la pièce avec le coffre, trouvez la **note « Bases des chiffres »** sur une étagère et le **porte-documents « Code des chiffres égyptiens »**. Ces deux éléments sont vos références principales.
  </Step>
  <Step num="2" title="Comprendre les bases des chiffres">
    Examinez la note « Bases des chiffres ». Ce document illustre les symboles utilisés dans les chiffres égyptiens anciens et leurs valeurs numériques correspondantes. Portez une attention particulière aux symboles qui sont entourés ou mis en évidence, car ce sont ceux dont vous aurez besoin pour le coffre.
  </Step>
  <Step num="3" title="Compter et convertir">
    Référez-vous au porte-documents « Code des chiffres égyptiens ». Sur ce document, vous verrez une série de symboles de chiffres égyptiens. Votre tâche consiste à compter combien de fois chaque symbole spécifique (tel qu'identifié dans la note « Bases des chiffres ») apparaît. Le décompte de chaque symbole, dans l'ordre, formera les chiffres du code. Par exemple, si un symbole représentant '3' apparaît une fois, un symbole représentant '2' deux fois, etc., vous formerez le code.
  </Step>
  <Step num="4" title="Saisir le code">
    Une fois que vous avez déterminé le code à quatre chiffres, approchez-vous du coffre aux chiffres égyptiens et saisissez la combinaison. Le code correct des **chiffres égyptiens** est **3262**.
  </Step>
</Steps>
