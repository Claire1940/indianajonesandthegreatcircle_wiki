<Section icon={Search} title="Himalaya : La Machine Chiffrement Kummetz">

La section Himalaya présente une énigme mécanique unique plutôt qu'un coffre standard. Pour progresser dans l'histoire principale à Kummetz, vous devez déverrouiller une porte scellée dans le Salon des Officiers à l'aide d'une machine à chiffrer. C'est le seul **code de coffre d'Indiana Jones et le Grand Cercle** obligatoire pour la progression de l'histoire dans cette région.

<Steps>
  <Step num="1" title="Localiser le Code du Capitaine">
    Trouvez le presse-papiers dans le salon avec les lettres **JÖGM**. C'est votre clé chiffrée.
  </Step>
  <Step num="2" title="Collecter les Roues de Code">
    Rassemblez les quatre disques métalliques dispersés dans la pièce. Chaque disque contient un ensemble unique de lettres.
  </Step>
  <Step num="3" title="Configurer la Machine à Chiffrer">
    Insérez les roues dans la machine. La roue 1 a 'J', la roue 2 a 'Ö', la roue 3 a 'G', et la roue 4 a 'M'.
  </Step>
  <Step num="4" title="Extraire le Mot de Passe">
    Faites tourner la poignée pour aligner **JÖGM**. La machine révélera le mot déchiffré : **BLUT**.
  </Step>
  <Step num="5" title="Consulter le Tableau">
    Vérifiez le tableau de correspondance sur la table. Le mot **BLUT** correspond au code numérique **3666**.
  </Step>
</Steps>

<Callout type="success" title="Progression de l'Histoire">
Entrer **3666** sur le clavier déverrouillera la lourde porte, permettant à Indy de progresser vers le sommet de la montagne.
</Callout>

</Section>
