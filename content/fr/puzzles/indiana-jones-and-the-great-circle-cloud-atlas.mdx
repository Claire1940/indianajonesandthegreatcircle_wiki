<Section icon={Target} title="Déchiffrer les Nombres du Cloud Atlas">
Le cœur du puzzle Cloud Atlas réside dans la traduction d'une séquence de nombres d'un morceau de papier en un code basé sur les nuages correspondant. Chaque type de nuage se voit attribuer un nombre de 0 à 9. Votre tâche consiste à faire correspondre les nombres trouvés sur le papier à leurs types de nuages respectifs, ce qui se traduit ensuite par une entrée numérique pour le coffre-fort.

Le papier que vous trouverez affichera une série de nombres à quatre chiffres, tels que "1310", "910", "1610" et "1410". Ces nombres ne sont pas des entrées directes ; ils correspondent plutôt à des types de nuages spécifiques qui, à leur tour, représentent les chiffres réels dont vous avez besoin.

<Callout type="warning" title="Erreur Courante">
Ne saisissez pas directement les nombres du papier dans le coffre-fort. Ce sont des codes de référence pour les types de nuages, pas la solution finale du puzzle. Reportez-vous toujours au mappage nuage-nombre.
</Callout>

Voici une ventilation du mappage nuage-nombre typique que vous rencontrerez :

| Type de Nuage | Numéro Assigné | Description |
|:--------------|:---------------|:------------|
| Cirrus        | **0**          | Nuages élevés et effilochés. |
| Stratus       | **1**          | Nuages bas et stratifiés. |
| Cumulus       | **2**          | Nuages cotonneux et duveteux. |
| Nimbostratus  | **3**          | Nuages sombres et producteurs de pluie. |
| Altocumulus   | **4**          | Nuages de niveau moyen, fragmentés. |
| Cirrocumulus  | **5**          | Nuages élevés et ondulés. |
| Cumulonimbus  | **6**          | Grands nuages d'orage imposants. |
| Stratocumulus | **7**          | Nuages bas et grumeleux. |
| Altostratus   | **8**          | Nappes grises de niveau moyen. |
| Lenticulaire  | **9**          | En forme de lentille, souvent près des montagnes. |

Vous devez croiser les nombres de votre papier avec ce mappage pour obtenir la séquence correcte pour le coffre-fort. Par exemple, si "1310" fait référence à "Cirrus" (qui correspond à 0), alors le premier chiffre de votre code de coffre-fort est 0.

<Section icon={Sparkles} title="Résolution du Puzzle du Coffre-Fort : Étape par Étape">
Une fois le mappage nuage-nombre compris, l'étape suivante consiste à saisir la séquence correcte dans le coffre-fort. C'est là que tout votre travail d'investigation se réunit pour déverrouiller le trésor qu'il contient.

<Steps>
  <Step num="1" title="Localiser le Coffre-Fort">
    Assurez-vous d'avoir le coffre-fort en votre possession et d'être prêt à interagir avec lui. Il présentera un pavé numérique.
  </Step>
  <Step num="2" title="Identifier le Papier Numérique">
    Trouvez le morceau de papier qui contient la séquence de nombres à quatre chiffres. C'est votre principal indice pour le puzzle.
  </Step>
  <Step num="3" title="Mapper les Références de Nuages aux Chiffres">
    Examinez attentivement chaque nombre à quatre chiffres sur le papier. Pour chaque nombre, déterminez à quel type de nuage il fait référence. Ensuite, utilisez le tableau de mappage nuage-nombre (comme indiqué dans la section précédente) pour obtenir le chiffre réel pour le coffre-fort. Par exemple, si le papier indique "1310" et que cela fait référence au nuage "Cirrus", votre premier chiffre est **0**. Si un autre nombre fait référence à "Cumulonimbus", votre chiffre suivant est **6**.
  </Step>
  <Step num="4" title="Entrer la Séquence">
    Entrez la séquence dérivée de chiffres uniques (0-9) dans le coffre-fort. Le puzzle nécessite généralement une séquence de quatre chiffres uniques.
  </Step>
  <Step num="5" title="Réclamer Votre Récompense">
    Une fois la séquence correcte entrée, le coffre-fort s'ouvrira, révélant son contenu et vous permettant de progresser davantage dans *Indiana Jones et le Grand Cercle*.
  </Step>
</Steps>
