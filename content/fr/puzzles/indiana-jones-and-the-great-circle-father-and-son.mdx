<Section icon={Target} title="Guide étape par étape : Père et Fils">

Pour réussir l'objectif **indiana jones and the great circle père et fils**, suivez ces étapes précises. Assurez-vous d'avoir nettoyé la zone immédiate de tout garde en patrouille, car le son de la serrure mécanique peut occasionnellement alerter les ennemis dans les sections axées sur l'infiltration.

<Steps>
  <Step num="1" title="Localiser la note en cursive">
    Entrez dans le bureau et cherchez un pupitre éclairé par une lampe à abat-jour vert. La note est placée en évidence sur le buvard. Interagissez avec elle pour passer en mode "Focus".
  </Step>
  <Step num="2" title="Identifier les lettres clés">
    Parcourez le texte pour trouver les majuscules 'I' et 'R'. Indy fera un commentaire sur "l'écriture inhabituelle". Cela confirme que vous regardez le bon indice pour le premier chiffre.
  </Step>
  <Step num="3" title="S'approcher du coffre-fort">
    Le coffre-fort est situé derrière un tableau ou à l'intérieur d'une bibliothèque à proximité. Déplacez le curseur d'interaction sur le cadran pour commencer à saisir le code.
  </Step>
  <Step num="4" title="Saisir le code 5837">
    Tournez le cadran dans le sens des aiguilles d'une montre pour le 5, dans le sens inverse pour le 8, dans le sens des aiguilles d'une montre pour le 3, et dans le sens inverse pour le 7. Vous entendrez un clic mécanique distinct pour chaque saisie correcte.
  </Step>
  <Step num="5" title="Récupérer l'artefact">
    Ouvrez la porte et prenez l'objet à l'intérieur. Cela déclenchera une courte cinématique ou une séquence de dialogue concernant le père d'Indy et le mystère plus large du Great Circle.
  </Step>
</Steps>

<Callout type="warning" title="Erreurs de saisie">
Si vous saisissez un chiffre de manière incorrecte, vous devez quitter complètement l'interaction pour réinitialiser le cadran. Le jeu ne permet pas d'effacer un seul chiffre une fois que le cadran est passé à la position suivante.
</Callout>
