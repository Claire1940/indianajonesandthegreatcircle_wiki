<Section icon={Target} title="Deciphering the Cloud Atlas Numbers">
The core of the Cloud Atlas puzzle lies in translating a sequence of numbers from a piece of paper into a corresponding cloud-based code. Each cloud type is assigned a number from 0 to 9. Your task is to match the numbers found on the paper to their respective cloud types, which then translates into a numerical input for the lockbox.

The paper you find will display a series of four-digit numbers, such as "1310", "910", "1610", and "1410". These numbers aren't direct inputs; rather, they correspond to specific cloud types that, in turn, represent the actual digits you need.

<Callout type="warning" title="Common Mistake">
Do not directly input the numbers from the paper into the lockbox. These are reference codes for cloud types, not the final puzzle solution. Always refer back to the cloud-to-number mapping.
</Callout>

| Cloud Type | Assigned Number | Description |

| Stratus    | **1**           | Low, layered clouds. |
| Cumulus    | **2**           | Fluffy, cotton-like clouds. |
| Nimbostratus | **3**         | Dark, rain-producing clouds. |
| Altocumulus | **4**          | Mid-level, patchy clouds. |
| Cirrocumulus | **5**         | High, rippled clouds. |

You need to cross-reference the numbers from your found paper with this mapping to get the correct sequence for the lockbox. For example, if "1310" refers to "Cyrus" (which corresponds to 0), then the first digit of your lockbox code is 0.
